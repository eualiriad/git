Entendendo Comandos Git

Projeto Criado:

- Terminal > Git Bash Vamos dar os comandos: $ ls -a (vai listar todos os arquivos e diretórios da pasta) $ git init
$ git status (vai nos falar os status do nosso arquivo, se sofreu ou não alterações.) $ git add . (vai adicionar as alterações) $ git commit - m'nome que quiser' (pra comitar as alterações)

DIZER SUA CONTA PRO GIT:

git config --global user.name "name" git config --global user.email "email@email.com"

Criando Chave SSH

Colocar no terminal: ssh-keygen -t ed25519 -C "seu_email@dominio.com" Na primeira linha o terminal avisa que está gerando as suas chaves, privada e pública com ed25519. Já na segunda linha está te perguntando onde você quer salvar as suas chaves SSH, entre parênteses e ainda na segunda linha o terminal tem uma sugestão do caminho e nome do arquivo onde será guardada a chave gerada. Digite uma senha se quiser, se não, aperte ENTER. - Ao final teremos: Senha pública (usaremos no GITHUB) Senha privada (mantenha em sigilo)

Cadastrando chave no GITHUB

No menu lateral esquerdo clique em SSH and GPG keys e cadastre.

Vinculando repositóro para a nuvem.

Procure por 'New' em GITHUB > 'Criar Repositório' Siga os passos apresentados. Após isso tudo feito.

Utilize $ git push após o $ git commit -m'name' Assim irá sempre salvar na nuvém após atualizar seu projeto.

GITHUB Pages

Dentro do seu code vá em 'settings' > 'Pages' Selecione 'main' e 'root' e pronto Link gerado Se der erro ao abrir é porque ainda está processando!
